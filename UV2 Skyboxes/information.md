# Skyboxes

This folder contains skybox textures extracted from resource packs to be ported for use in Miniblox

## Contents

* Skybox textures grouped by resource pack.
* 6 images of the skybox (cubemap) which can be converted to a 3d skybox used in game

## Credits

All skybox textures belong to their respective resource pack creators. This folder simply ports and organizes the files for use with miniblox and does not claim ownership of the original assets.
